Class

对象容器

  • 顺序容器
    • 范型容器类:ArrayList (–> 容器的类型<元素的类型>)
    • ArrayList的相关操作 (String)
      1.get.(index)
      2.add(String),add(String,index)
      3.remove(index)
      4.toArray(String[] a)
  • 对象数组
    • String[] array,Class[] array
    • 对象数组中的每个元素都是对象的管理者而非对象本身
    • for-each循环
      for( Class class : array) {
      System.out.println(class.get()); // print(class)
      }
  • 集合容器
    • HashSet
    • 类似数学中的set,在集合当中没有重复的元素，重复元素只记录最后一次
    • ArrayList和HashSet这类容器可以用println直接输出
      结果类似:
      [first,second,third,…]
    • toString方法
  • 散列表
    • HashMap<Key, Type> // 包裹类型,必须是对象,不能是基本类型
    • 相关操作
      1.put(key, value) // 重复填入的key会被新的覆盖
      2.get(key)
      3.containsKey(key)
      4.KeySet().size() // KeySet()将所有的key做成一个hashset
      5.println(hashmap)
      6.for-each循环
      HashMap<Integer, String> hm = new HashMap<Integer, String>()
      …
      for ( Integer i : hm.KeySet()) {
      String s = hm.get(i);
      System.out.println(s);
      }
